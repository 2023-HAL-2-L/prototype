## 命名規則

参考資料:
<https://qiita.com/Ted-HM/items/7dde25dcffae4cdc7923>

### 共通項目

- **ファイル名は全て一目で意味の通じる英語で記述する**

### HTML

- ファイル名、クラス、id名は全てケバブケースで記述する

### SQL

- テーブル名、カラム名は全てスネークケースで記述する

### その他のプログラムファイル

- ファイル名は全てロワーキャメルケースで記述する
